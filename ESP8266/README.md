# ESP8266
## Pin Map
TXD/RXD -> PMS7003(boud=9600)  
GPIO5/GPIO4 -> RS485(boud=115200)  
GPIO0 -> Download/Normal(Don't set it!!)  
GPIO2 -> Mode(AP for setting/STA for station)  
ADC -> Battery  
REST -> GPIO16  
GPIO12 -> Green LED  
GPIO13 -> Red LED  

## How to download into ESP8266
TXD/RXD -> USB to TTL  
GPIO0 -> GND  
REST -> 3.3V -> press down -> GND -> release -> 3.3V  
Download it!!
